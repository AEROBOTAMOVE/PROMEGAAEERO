# AERO Metals Bot v5.5c 🥇🥈

Автономен сигнален бот за **ЗЛАТО (XAUUSD) + СРЕБРО (XAGUSD)**, LONG + SHORT.
На всеки 5 минути, докато пазарът е отворен (неделя вечер → петък вечер по
CME; уикендът почива): свежи данни (Yahoo барове + **спот в реално
време**), confluence сигнал от **макро + ценова структура** (проверен на
няколко таймфрейма), и в Телеграм — компактни карти с **готови числа за
брокера** (вход/ТП1/ТП2/ТП3/СТОП от живия спот) + ред
**„ВЛИЗАЙ: ДА/ВНИМАТЕЛНО/НЕ“** от калибрирани статистики.

## Какво прави различно
- **Нула смятане**: всички нива са готови числа от спот цената в секундата
  на пращане — слагаш ги при брокера и толкова.
- **Следене докрай**: ✅ТП1 („премести стопа“) → ✅✅ТП2 → 🏆ТП3 / 🛑СТОП —
  с часа на РЕАЛНИЯ удар, по живия спот (≤5 мин денем, ≤10-15 нощем), гап-честно.
- **Казва дали да влизаш**: пресен/застоял клас, US-щит (8:25–9:15 Ню Йорк),
  стоп-пазач (2 стопа/посока/ден), ре-влизане само където данните печелят.
- **Контекст**: седмичен + дневен анализ в картата; събитие с висока
  важност пали щита автоматично.
- **Вечерна равносметка в 21:00** + статус-карта при поискване (Run workflow → status: yes).
- **Брониран**: пощенска кутия (съобщение не се губи при срив/429), самолекуващо
  се състояние, selftest-бариера при всяко качване (tests.yml), спот-резерва (PAXG),
  🤖 одит-робот 3×/ден.

## Качване
Стъпките (ново публично repo + бот + секрети + крон) са във файла
**КАК_СЕ_КАЧВА_БОТА.txt** (извън тази папка). Трите workflow-а вече са тук в
`.github/workflows/` — нищо не се копира ръчно, само издърпваш папката в repo-то.

Секрети (Settings → Secrets → Actions): `TELEGRAM_TOKEN`, `TELEGRAM_CHAT_ID`.
По желание (Settings → Variables): `BALANCE`, `RISK_PCT`.

## Честно (не се крие)
- Backtest = минало, предимно бичи период; **не гарантира бъдеще.** Ръбът се
  доказва само от форуърд-тест на живо.
- Yahoo баровете са с ~10-15 мин закъснение → решението за вход стъпва на
  тях (спот-цените на картата са реално време; затова са ОТМЕСТВАНИЯТА).
- Сигналите между таймфреймовете се припокриват (едно макро ядро) — затова
  казваме „макро + ценова структура“, не „N независими гласа“.
- **Хартия/малък размер първо. Не е финансов съвет** — човекът решава.
